## Guidelines
- Create a new branch for each task/change you are working on.
- All branches should be created from `main` branch.
- Branch name should be in the following format: `feature/<branch-name>`, `bugfix/<branch-name>`, `hotfix/<branch-name>`.
- Commit messages should be in the following format: `#<issue-number> <commit-message>`.
